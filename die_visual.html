Dice Rolling Simulator

This page simulates the results of rolling any number of dice, of any number of sides, any number of times. Enter the number of sides you want your dice to have, then the number of dice you want to roll, then the number of times you want to roll those dice. For example, if you want to see the distribution of results when rolling five D6 dice many times, you could enter "6" for the number of sides, "5" for the number of dice, and "1000000" for the number of times.
